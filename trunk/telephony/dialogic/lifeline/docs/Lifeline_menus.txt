System box (0***)

Plays system stats
	Enter system code to access (see Main Menu)

Main Menu
1) Change box properties
	****) back to menu
	Enter box number
		1) Listen to box properties
		3) Restore box properties
		4) Change reminder
			1) Toggle Reminder
			#) Clear Reminder
			*) Back to menu
		7) Change administrative box for this box
		   (not available from admin box)
			****) Back to menu
			####) Clear administrative box
			Enter new admin box 
			(box number must refer to valid admin box)
				Reenter new admin box
		8) Change box type
		   (not available from admin box)
			1) Toggle type
			*) Return to menu
		9) Activate or deactivate a box
		#) Change security code for a box
			****) Back to menu
			####) Clear security code
			Enter new security code
				Reenter new security code
		*) Save changes and go back to system menu

	Press a key to return to menu

	Press a key to return to menu
